## English

### About this Repository
Welcome to my networking portfolio! This repository will contain a progressive collection of network infrastructure labs using **Huawei VRP** (Versatile Routing Platform) and the **eNSP** simulator.

The goal of these projects is to demonstrate hands-on skills in network design, configuration, and troubleshooting, scaling from basic routing to advanced network services and security.

### Lab Index
* **[Lab 01: Basic Routing and Switching](./01-basic-routing-and-switching)**

### Repository Structure
Each lab folder follows a standardized structure for easy navigation:
* `ensp-project/`: The eNSP `.topo` files.
* `configs/`: Text files with the exported configurations from each device.
* `images/`: Screenshots of topologies, configurations, and testing.
* `docs/`: Documentation available in English (`en/`) and Spanish (`es/`).

### Requirements
To open and run the topology files, you will need:
* **Huawei eNSP** (V100R003C00SPC100 or compatible).
* VirtualBox (compatible version required by eNSP).

## Español

### Sobre este Repositorio
¡Bienvenido a mi portafolio de redes! Este repositorio contendrá una colección progresiva de prácticas de infraestructura de red utilizando **Huawei VRP** (Versatile Routing Platform) y el simulador **eNSP**.

El objetivo de estos proyectos es demostrar habilidades prácticas en diseño, configuración y resolución de problemas de red, escalando desde el enrutamiento básico hasta servicios de red avanzados y seguridad.

### Índice de Prácticas
* **[Lab 01: Enrutamiento Básico y Switching](./01-basic-routing-and-switching)**

### Estructura del Repositorio
Cada carpeta de práctica sigue una estructura estandarizada para facilitar la navegación:
* `ensp-project/`: Los archivos `.topo` de eNSP.
* `configs/`: Archivos de texto con las configuraciones exportadas de cada equipo.
* `images/`: Capturas de pantalla de topologías, configuraciones y pruebas.
* `docs/`: Documentación disponible en Inglés (`en/`) y Español (`es/`).

### Requisitos
Para abrir y ejecutar los archivos de topología, necesitarás:
* **Huawei eNSP** (V100R003C00SPC100 o compatible).
* VirtualBox (versión compatible requerida por eNSP).
